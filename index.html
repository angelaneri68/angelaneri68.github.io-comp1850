HTML tags

This page describes the following HTML tags

  • Blockquote
  • Code
  • Definition List
  • Select
  • Pre

Blockquote

Definition
The blockquote represents a section that is quoted from another source
Semantics
Blockquote is a semantical tag
Type
Block tag
Attributes
Use Global Events Attributes and cite attribute, that is the source of citation
Example

Code

<blockquote 
cite="https://glnews.ca/news/5057/van-3-life/">
Vancouver ranks third best in the world for 
quality of life
</blockquote>

Result

Vancouver ranks third best in the world for quality of life

Code

Definition
The code defines a piece of computer code
Semantics
Code is a phrase tag
Type
Inline tag
Attributes
Does not have special attributes. Use Global Events Attributes
Example

Code

<code>
This is a piece of code
<code>

Result

This is a piece of code

Definition List

Definition
It's a list where entries are listed like in a dictionary or encyclopedia. The definition list is the ideal way to present a glossary, list of terms, or other name/value list.
Used with dt and dd tags, where dt is the term and dd is the description
Semantics
Definition List is a semantical tag
Type
Block tag
Attributes
Does not have special attributes. Use Global Events Attributes
Example

Code

<dl>
 <dt>Coffee </dt>
  <dd> black drink
  </dd>
 <dt>Milk</dt>
  <dd> white drink 
  </dd>
</dl>

Result

Coffee
black drink
Milk
white drink

Select

Definition
The select is used to create a drop-down list.It is a form control and can be used in a form to collect user input. The option tags inside the select element define the available options in the list
Semantics
Select is a semantical tag
Type
Inline tag
Attributes
Does not have special attributes. Use Global Events Attributes
Example

Code

<select>
<option value="morning">Morning</option>
<option value="afternoon">Afternoon</option> 
<option value="nigth">Night</option> 
</select>

Result

Pre

Definition
The pre tag defines preformatted text. Text in a pre element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.
Semantics
Pre is not a phrase tag
Type
Block tag
Attributes
Use Global Events Attributes and width attribute, that is the maximum number of characters per line. This attribute is not supported in HTML5.
Example

Code

<pre>
Following text is displayed
            as it was 
                      preformatted
</pre>

Result

 Following text is displayed
            as it was 
                      preformatted
