This page describes a tutorial for the use of opacity property to "brigth" images

  1. We need to set one or more images as background. In this case, image was set up in the class .backdrop
  2. Then, we need to set the attribute for opacity in an element in front of the image. In this example, I used the class .text
  3. To create the Ligthing up effect, it's possible to use hover event so luminosity will change when we hover the mouse over the object.
  4. In the class .text:hover I updated the color of text and opacity
  5. I used the class text in a <p> tag as an example, and a <div> tag to set up image background.

This tutorial was inspired by:

  • W3 scholl- CSS opacity
