Project Proposal

This page describes the plan for final project WebSite.

Theme

The theme of this website is an online resume.

Objectives

The main objective is detailing professional life in an interactive way.

It needs to be easy to use and recover information of projects that I have collaborated, my work experience, knowledge, achievements and education.

Target Audience

The target audience will be recruiters, prospective employes and HR analysts, located at Great Vancouver, or maybe in other Canadian cities, for remote work. Main language is English. These roles has the following characteristics:

  • Recruiters: Mainly women, Age: 25-35
  • HR analysts: Mainly women, Age: 30-40
  • Prospective employes: Composed by IT managers and Executives, mainly men (70/30), Age: 30-55

Description

As an online resume, this website needs to have at least the following information

  1. Who I am: A brief description of my personal, professional and educational characteristics, similar to a cover letter.This will be the home page.
  2. Skills and Capabilities: description of main skills, relating them to projects and certifications.
  3. Work history: Description of companies, roles, responsabilities and achievements
  4. Education: My education,trainnings,seminars, languages.
  5. Projects: Main projects I have worked with business results, some clients,achieved gols.
  6. Contact me: Email, linkedin
