WEB usability

This page describes web usability and accessibility for some web sites.

Competition

I have analised many personal web sites, and they differ in structure and content. Some of them focuses on portfolio, others personal characteristics. However for IT professionals, the most part of them details the technical skills.

Related to structure, it seems to be a trend not to use many pages. All data can be seem vertically, using scroll bar, maybe because they are easier to read on mobile devices. For example, Albino Tonnina uses a timeline to struct the information, what I really think that is very clear. In my opition it has too much animation on transitions. Although, he provides a menu with links to main topics. These ones use the same idea, but with less animation: Keith Himan and Pascal Vangermert. One of the best that I saw was this: Robby Leonard, that was certainly built for his target audience.

On the other site, structured as many pages and more traditional ones, can be very usefull for formal environments, as this example: Callie Schweitzer.

Usability

One website that I use regularly is Netflix. I really like movies and series so it is almost a survival matter!

Its main objective is selling on demand streaming to any kind of audience: children, youths, matures, males, females,... in many countries around the world, with different cultures and tastes.

So it needs to:

  1. be easy to use and find items that you want to see and
  2. estimulate any customer to see other items

It deploys these objectives using always same layout, with searching tools and items categorized in lists, but the items in each list and trailler suggestions are individually determined by the user's profile and usage, which clearly improves the user's curiosity.

Color usage

I like the collor pallete of this website Ellen Rilley. It is collorful and cheerful, neverthless is not informal. The contrast between the background and the text color facilitates the reading, and even the pictures are hormonalized with the background.

These are the main colors of the Ellen's site.

Layout

My favorite layout was Keith Himan. His website is very clear and easy to find information. He uses a timeline to ilustrate his profissional experience, what I have already said that I think that is a very good idea. The menu is always visible, so you can choose other information anytime. A "Top" button is always visible to, what is useful in this one-page layouts.

The use of two collumns is good to visualize much information keeping white-spaces. He uses collors in a more conventional way, and he restricts the fonts to 2 or 3 styles.So it is clean and stylish.
